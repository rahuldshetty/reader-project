
Modern-UI RSS desktop app to browse RSS feeds. 

Functionality:
--------------

- Add RSS feeds
- List RSS feeds and posts
    - Options: All Posts, Unread, Bookmarked, individual feed posts
- View RSS page content
- Notification Service
- Importing OPML files
- Regex filtering
- Bookmark/Fav a post

- Modern UI
- Thumbnail View or Lane View
- Settings Page
- Light Mode/Dark Mode
